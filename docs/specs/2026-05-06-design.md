# Specification: Liquor Store Clone & Upgrade

## Overview
Clone giao diện Liquor Store (furniture-psi-five.vercel.app) và nâng cấp với tech stack hiện đại, đảm bảo tính thẩm mỹ, hiệu năng (performance), và trải nghiệm người dùng (UX) chuẩn production.

## Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Components:** shadcn/ui (Radix UI)
- **State Management:** Zustand (Cart), TanStack Query (Data fetching)
- **Forms:** React Hook Form + Zod (Validation)
- **Testing:** Playwright (E2E), Vitest (Unit)

## Key Features
1. **Home:** Hero section, Featured Products, Categories, Testimonials.
2. **Product Listing:** Filter (by category), Sort, Search.
3. **Product Detail:** Image gallery, Product info, Add to cart.
4. **Cart:** Add/Remove/Update items, persistent storage.
5. **Checkout:** Mock process.
6. **Auth:** User Login/Register placeholders.
7. **Admin Dashboard (Simulated):** View, Add, Edit, Delete Products.

## Data Architecture
- **Products:** ID, Name, Category, Price, ImageURL, Description, Stock.
- **Categories:** ID, Name, Icon.
- **Cart:** Array of items (ProductID, Quantity).
