## About
Developed a high-performance, scalable e-commerce platform for liquor retail, built with Next.js 15 and TypeScript for optimal SEO and performance. Streamlined data fetching and caching with TanStack Query, while leveraging Zustand for robust global state management. Crafted a modern, accessible, and fully responsive UI using Tailwind CSS and shadcn/ui. Integrated secure authentication via NextAuth.js. Implemented an integrated real-time messaging system to facilitate seamless communication and direct customer support. Architected a scalable backend using Prisma ORM and a PostgreSQL database, powering a comprehensive admin dashboard that enables efficient, data-driven management of product catalogs, real-time inventory tracking, order fulfillment, and user accounts.
